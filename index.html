Welcome to the Shell Guessing Game!

By: Aaron Princely Enyetu

The game begins with 3 shells, one of which is concealing a ball. A random generator mixes the shells so that the user does not know where the ball is. User's should see three shells and three buttons and need to guess where the ball is "hidden". When the user clicks on one of the buttons, the ball's location is revealed. The user should also track the total number of wins, losses, and total games played are displayed.
